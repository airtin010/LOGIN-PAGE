## Estrutura do Projeto

```
LOGIN PAGE/
├── forgot.html
├── login.html
├── register.html
└── static/
    ├── index.css
    └── sub/
        ├── forgot.css
        ├── login.css
        └── register.css
```

- **login.html**: Página de login.
- **register.html**: Página de cadastro.
- **forgot.html**: Página de recuperação de senha.
- **static/index.css**: Estilos globais e tema escuro rosa.
- **static/sub/**: Estilos específicos para cada página.
